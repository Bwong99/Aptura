# Aptura 

**ACCESS WEBSITE [HERE](https://aptura-dysh.onrender.com/)**

Photography and creative photo sharing platform  

---

## 🚀 Features
- 🔄 **Real-time sync** PostgreSQL Database
- 🎨 **Interactive UI/UX** responsive design  
- 👥 **Live collaboration** through follows and post saves
  
---

## 🧰 Tech Stack
Backend:
- 🐍 Python - Programming Language
- 📁 Django - Web Framework and installation packages
- 🗄️ PostgreSQL - Database
- 📷 Cloudinary Static Files - For storing and serving media files

Frontend
- 🌐 HTML - Markup
- 🎨 CSS - Styling
- ⚡ JavaScript - Client-side interactivity

Website Deployment 
- 🎯 Render
